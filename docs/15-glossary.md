# Glossary

## Purpose

This page defines the key terms used across BRIMS documentation.

## How to Use This Page

Refer to this glossary when a BRIMS term has a specific operational meaning or when similar terms could be confused.

## Terms

### Arm

Placeholder: definition of a participant grouping within a project.

### Assay

Placeholder: definition of a laboratory analysis or linked assay record.

### Event

Placeholder: definition of a scheduled or completed participant activity.

### Manifest

Placeholder: definition of a specimen shipment or transfer record.

### Participant

Placeholder: definition of a person enrolled in a project.

### Project

Placeholder: definition of the top-level unit used to organise work in BRIMS.

### Site

Placeholder: definition of a physical or organisational location within a project.

### Specimen

Placeholder: definition of a biological sample tracked in BRIMS.

### Storage Designation

Placeholder: definition of the storage identifier used for project or specimen operations.

### Study

Placeholder: definition of a research investigation within a project.

### Subject ID

Placeholder: definition of the unique participant identifier generated by BRIMS.

### Status

Placeholder: definition of system states used to indicate workflow progress.
